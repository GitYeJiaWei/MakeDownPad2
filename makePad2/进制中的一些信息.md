#进制中的一些信息

```java
             if ((statusData[0] & 0x40) == 64) {
                  handler.sendEmptyMessage(3);
                  Log.d("getState", "打印机温度异常");
              } else {
                  Log.d("getState", "打印机温度正常");
              }

其中 0x40 是16进制数 对应的十进制数是 64，对应的二进制数是0100 0000；

0x40 中0x代表16进制，4对应二进制0100，0对应二进制0000；	

0100 0000 中的每一位根据2的次方数计算，从2的0次方到2的7次方，即0，2，4，8，16，32，64，128；

& 为与的意思，1&1=1，0&任何=0，0|1=1   所以(statusData[0] & 0x40) == 64的意思就是 1111 1111 & 0100 0000 =0100 0000
```
